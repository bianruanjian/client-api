基于控制台封装的打印组件，print.log(string content)


## 方法

| 名称  | 说明 | 参数    |
| ----- | ---- | ------- |
| log   | 日志 | content |
| info  | 提示 | content |
| warn  | 警告 | content |
| error | 错误 | content |

## 参数

| 名称    | 说明     | 值类型 | 默认值 | 是否必填 |
| ------- | -------- | ------ | ------ | -------- |
| content | 打印内容 | any    |        | 否       |
